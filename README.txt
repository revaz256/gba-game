Press SELECT at any time to go back to the start screen. 
Press START at the start screen to begin playing the game.
RULES: 
1. If you (the red bar) hit the white bar that goes down the screen, you lose.
2. If you avoid the white bar, it will start back at the top.
3. You can move the red bar left, right, up, and down using the LEFT, RIGHT, UP, and DOWN arrows on your keyboard.
